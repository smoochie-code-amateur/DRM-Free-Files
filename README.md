# DRM-Free/Offline Files for games

#### Piracy?
No, I don’t think so. The thing is, you’ll need to have an **original** copy of the game. Or rather, a **legal** one. It’s just that I personally find it annoying that, for example, a game is single-player but is tied to clunky launchers and a constant online connection. Nonsense. That’s why these files will help you get round it.

#### Where do they come from?
From cracks. True. But only these files, not the games themselves – it’s important to understand that.
